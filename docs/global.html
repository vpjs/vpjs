__VPJSBOOTSTRAP__ :Array

You can define the bootstrap via __VPJSBOOTSTRAP__, this must be array of functions. Each function will be called with 1 parameter, what is the vp.js core module Important, the bootstrap must be defined before loading the library!
Type:
  • Array
Source:
  • vp.js, line 14
Example
var __VPJSBOOTSTRAP__ = [function (Core) {
    Core.set();
}];

vpjs

Core vp.js interface The behavior can be deferent, base on the first parameter
Source:
  • vp.js, line 1221
Examples
//String
vpjs('event'); // will publish a event
vpjs('event', {}); // will publish a event with data
vpjs('event', function () {}); //@todo
vpjs('event', {}, function () {}); //@todo
vpjs('>event'); //$/events~pub
vpjs('<event', function(){}); //$/events~sub
//String contains a path
vpjs('ns/module', function() {}); //$/amd~def
//Array
vpjs(['Module']); //$/amd~def
vpjs(['Module'], function () {}); //$/amd~def
//Function
vpjs(function(){}); //$/amd~def
