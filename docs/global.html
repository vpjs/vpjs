__VPJSBOOTSTRAP__ :Array

You can define the bootstrap via __VPJSBOOTSTRAP__, this must be array of functions. Each function will be called with 1 parameter, what is the vp.js core module Important, the bootstrap must be defined before loading the library!
Type:
  • Array
Source:
  • vp.js, line 6
Example
var __VPJSBOOTSTRAP__ = [function (vp) {
    vp.set();
}];
