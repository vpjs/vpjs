__VPJSBOOTSTRAP__ :Array

You can define the bootstrap via __VPJSBOOTSTRAP__, this must be array of functions. Each function will be called with 1 parameter, what is the vp.js core module Important, the bootstrap must be defined before loading the library!
Type:
  • Array
Source:
  • vp.js, line 14
Example
var __VPJSBOOTSTRAP__ = [function (vp) {
    vp.set();
}];

__VPJSNS__ :string

You can change the namespace via __VPJSNS__, all core events will start with you defined namespace and not with 'vp'

  • string

vp

Core vp.js interface The behavior can be deferent, base on the first parameter
Source:
  • vp.js, line 1352
Examples
//String
vp('event'); // will publish a event
vp('event', {}); // will publish a event with data
vp('event', function () {}); //@todo
vp('event', {}, function () {}); //@todo
vp('>event'); //$/events~pub
vp('<event', function(){}); //$/events~sub
//String contains a path
vp('ns/module', function() {}); //$/amd~def
//Array
vp(['Module']); //$/amd~def
vp(['Module'], function () {}); //$/amd~def
//Function
vp(function(){}); //$/amd~def
