<inner> sub(evnt, subscriber, scope, thisArg)

Subscribe to a event valid subscribe events 'x' or 'somelongname' or 'a123' the first character of a event must be a-z, a event must only contain only small characters or numbers 'x.x' you can use a '.' to name space the event 'x.*' you can use a '*' to subscribe to all events in a name space '*' subscribe to all events invalid subscribe events '1a' the first character can't be a number 'a.' name space can't be empty 'a#$' a event can't have other characters than a-z or 0-9
Parameters:
Name Type Argument Description
evnt String The event where you want to subscribe
subscriber function The subscriber to the events
scope * <optional>
 Scope can be used to only subscribe to events that are in that scope
thisArg * <optional>
 The `this` scope of the subscriber
