Has the first parameter the needed value? By a string and array it will search in the string for the needed value By a number it will check of the first parameter is bigger or equal than the needed number By a object it will check of the object has needed property By everything else it will compare the 2 both parameters and if they are the same it will return true.

<static> merge(target, items, deep) → {Array|Object|String}

Will merge all parameters based on the type of the first parameter. If the first parameter is a Array then it will treat all other parameters as a Array by using Y.toArray If the first parameter is a Object it will extend this object with the other objects All other types will be merge to 1 single string IF the first parameter is a Object and the last parameter is a boolean it will do a deep merge of all the object objects, meaning that if a property contains a object from the first parameter it will also merge that
Parameters:
Name Type Argument Default Description
target Array | Object | *
items * <repeatable>
deep Boolean <optional>
 false
Source:
  • vp.js, line 482
Returns:
Type
Array | Object | String
Example
merge({x:1}, {y:1}) //will result in {x:1, y:1}
merge([1], [2]) //will result in [1,2]
merge({x:{y:2}}, {x:{z:1}}, true)// will result in  {x:{y:2,z:1}}
